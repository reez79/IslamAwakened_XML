# IslamAwakened_XML
IslamAwakened XML database reader

G.Waleed Kavalec is the founder of IslamAwakened.com which hosts numerous Quran translations.
https://islamawakened.com/

The translations have been made available for download in XML format (ia_all.xml).  
The ia_all.zip file shared on this github page contains the ia_all.XLM file current as of 2026-May-29.

This program serves to interpret, filter, search, and read the XLM database.
If you can run the Python, go for it.  Most will need to download an executable, find that on the "releases" page.

USAGE:
Place both files (ia_all.XML & QuranVerseExplorer.exe in a common windows folder.
